non-inertia frame 非惯性系中的牛顿定律

Created2026-05-27|Updated2026-05-27
|Post Views:

定律

对于任何非惯性系，均有以下式成立, 只是“附加项”的具体形式会随参考系的运动情况而不同。
一般形式（平动 + 转动）是：

a惯=a相+aO′+ω˙×r′+ω×(ω×r′)⏟牵连加速度  +  2ω×v相⏟科里奥利\boxed{\mathbf{a}_{\text{惯}} = \mathbf{a}_{\text{相}} + \underbrace{\mathbf{a}_{O'} + \boldsymbol{\dot{\omega}} \times \mathbf{r}' + \boldsymbol{\omega} \times (\boldsymbol{\omega} \times \mathbf{r}')}_{\text{牵连加速度}} \;+\; \underbrace{2 \boldsymbol{\omega} \times \mathbf{v}_{\text{相}}}_{\text{科里奥利}}} a惯​=a相​+牵连加速度aO′​+ω˙×r′+ω×(ω×r′)​​+科里奥利2ω×v相​​​​

应用一:

在匀速转动的圆盘上建立参考系，这是一个典型的非惯性系, 圆盘绕中心轴以恒定角速度 ω\omegaω 转动.

a惯=(r¨−rθ˙2)er+(rθ¨+2r˙θ˙)eθ\mathbf{a}_{\text{惯}} = (\ddot{r} - r\dot{\theta}^2)\mathbf{e}_r + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\mathbf{e}_\theta a惯​=(r¨−rθ˙2)er​+(rθ¨+2r˙θ˙)eθ​

a相=(r¨−rϕ˙2)er+(rϕ¨+2r˙ϕ˙)eθ\mathbf{a}_{\text{相}} = (\ddot{r} - r\dot{\phi}^2)\mathbf{e}_r + (r\ddot{\phi} + 2\dot{r}\dot{\phi})\mathbf{e}_\theta a相​=(r¨−rϕ˙​2)er​+(rϕ¨​+2r˙ϕ˙​)eθ​

非惯性系中的原点没有平动加速度: aO′=0\mathbf{a}_{O'} = 0aO′​=0
匀速转动没有角加速度: ω˙×r′=0\boldsymbol{\dot{\omega}} \times \mathbf{r}' = 0ω˙×r′=0
附加项一: ω×(ω×r′)=ω2rer\boldsymbol{\omega} \times (\boldsymbol{\omega} \times \mathbf{r}') = \omega^2r \mathbf{e}_rω×(ω×r′)=ω2rer​

v相=r˙er+rϕ˙eθ\mathbf{v}_{\text{相}} = \dot{r} \mathbf{e}_r + r\dot{\phi} \mathbf{e}_\theta v相​=r˙er​+rϕ˙​eθ​

附加项二: 2ω×v相=−2ωrϕ˙ er+2ωr˙ eϕ2 \boldsymbol{\omega} \times \mathbf{v}_{\text{相}} = -2 \omega r\dot{\phi}\,\mathbf{e}_r + 2 \omega\dot{r}\,\mathbf{e}_\phi2ω×v相​=−2ωrϕ˙​er​+2ωr˙eϕ​

综上, 有:

a惯=a相  +  (−2rϕ˙ω−rω2)er+(2r˙ω)eθ⏟附加加速度\mathbf{a}_{\text{惯}} = \mathbf{a}_{\text{相}} \; + \; \underbrace{(-2r\dot{\phi}\omega - r\omega^2)\mathbf{e}_r + (2\dot{r}\omega)\mathbf{e}_\theta}_{\text{附加加速度}} a惯​=a相​+附加加速度(−2rϕ˙​ω−rω2)er​+(2r˙ω)eθ​​​

在惯性系中有:

ma惯=F外力m\mathbf{a}_{\text{惯}} = \mathbf{F}_{\text{外力}} ma惯​=F外力​

在非惯性系中观察, 有:

ma相=F外力+mrω2er⏟惯性离心力  +  2mωrϕ˙er  −  2mωr˙eθ⏟科里奥利力m\mathbf{a}_{\text{相}} = \mathbf{F}_{\text{外力}} + \underbrace{m r\omega^2 \mathbf{e}_r}_{\text{惯性离心力}} \; + \; \underbrace{2m\omega r\dot{\phi} \mathbf{e}_r \; - \; 2m\omega \dot{r} \mathbf{e}_\theta}_{\text{科里奥利力}} ma相​=F外力​+惯性离心力mrω2er​​​+科里奥利力2mωrϕ˙​er​−2mωr˙eθ​​​

例: 一物体在某一位置随圆盘转动, 某一时刻突然圆盘变光滑, 求圆盘视角非惯性系下, 极坐标系下的轨迹参数方程

目标: 求r(t)r(t)r(t) 和 ϕ(t)\phi(t)ϕ(t) 参数方程

初始位置: r(0)=r0r(0) = r_0r(0)=r0​, ϕ(0)=ϕ0\phi(0) = \phi_0ϕ(0)=ϕ0​

初始速度: t=0t=0t=0 时，物体在转动系中相对静止，即 r˙(0)=0,  ϕ˙(0)=0\dot{r}(0)=0,\;\dot{\phi}(0)=0r˙(0)=0,ϕ˙​(0)=0。

把相对加速度在极坐标下表达式代入, 无外力:

(r¨−rϕ˙2)er+(rϕ¨+2r˙ϕ˙)eθ=mrω2er⏟惯性离心力  +  2mωrϕ˙er  −  2mωr˙eθ⏟科里奥利力(\ddot{r} - r\dot{\phi}^2)\mathbf{e}_r + (r\ddot{\phi} + 2\dot{r}\dot{\phi})\mathbf{e}_\theta = \underbrace{m r\omega^2 \mathbf{e}_r}_{\text{惯性离心力}} \; + \; \underbrace{2m\omega r\dot{\phi} \mathbf{e}_r \; - \; 2m\omega \dot{r} \mathbf{e}_\theta}_{\text{科里奥利力}} (r¨−rϕ˙​2)er​+(rϕ¨​+2r˙ϕ˙​)eθ​=惯性离心力mrω2er​​​+科里奥利力2mωrϕ˙​er​−2mωr˙eθ​​​

径向：

r¨−rϕ˙2=rω2+2ωrϕ˙\ddot{r} - r\dot{\phi}^2 = r\omega^2 + 2\omega r\dot{\phi} r¨−rϕ˙​2=rω2+2ωrϕ˙​

横向：

rϕ¨+2r˙ϕ˙=−2ωr˙r\ddot{\phi} + 2\dot{r}\dot{\phi} = -2\omega\dot{r} rϕ¨​+2r˙ϕ˙​=−2ωr˙

这是一个二阶非线性耦合常微分方程组

  1. 方程组的数学类型
    原方程组为：

{r¨−rϕ˙2=rω2+2ωrϕ˙rϕ¨+2r˙ϕ˙=−2ωr˙\begin{cases} \ddot{r} - r\dot{\phi}^2 = r\omega^2 + 2\omega r\dot{\phi} \\ r\ddot{\phi} + 2\dot{r}\dot{\phi} = -2\omega\dot{r} \end{cases} {r¨−rϕ˙​2=rω2+2ωrϕ˙​rϕ¨​+2r˙ϕ˙​=−2ωr˙​

这是一个关于未知函数 r(t)r(t)r(t) 和 ϕ(t)\phi(t)ϕ(t) 的自治（autonomous）非线性常微分方程组（因为方程中不显含时间 ttt）。由于最高阶导数是二阶，我们可以将其化为一阶系统的标准形式。令

y1=r,y2=ϕ,y3=r˙,y4=ϕ˙,y_1 = r,\quad y_2 = \phi,\quad y_3 = \dot{r},\quad y_4 = \dot{\phi}, y1​=r,y2​=ϕ,y3​=r˙,y4​=ϕ˙​,

则原方程可写为 一阶动力系统：

ddt(y1y2y3y4)=(y3y4y1(y4+ω)2−2y3y1(y4+ω)).\frac{d}{dt} \begin{pmatrix} y_1 \\ y_2 \\ y_3 \\ y_4 \end{pmatrix} = \begin{pmatrix} y_3 \\ y_4 \\ y_1(y_4 + \omega)^2 \\ -\frac{2y_3}{y_1}(y_4 + \omega) \end{pmatrix}. dtd​​y1​y2​y3​y4​​​=​y3​y4​y1​(y4​+ω)2−y1​2y3​​(y4​+ω)​​.

这是 y˙=f(y)\dot{\mathbf{y}} = \mathbf{f}(\mathbf{y})y˙​=f(y) 的形式，属于四维相空间中的连续动力系统。

  1. 初始/边界条件
    对二阶方程组，要确定唯一解通常需要 初值条件（柯西问题），即指定 t=0t=0t=0 时的函数和一阶导数值：

r(0)=r0,ϕ(0)=ϕ0,r˙(0)=0,ϕ˙(0)=0.r(0) = r_0,\quad \phi(0) = \phi_0,\quad \dot{r}(0) = 0,\quad \dot{\phi}(0) = 0. r(0)=r0​,ϕ(0)=ϕ0​,r˙(0)=0,ϕ˙​(0)=0.

这四个条件正好对应四个一阶变量 (y1,y2,y3,y4)(y_1,y_2,y_3,y_4)(y1​,y2​,y3​,y4​) 的初值，构成适定的初值问题。理论上也可以提边界条件（如指定 t=0t=0t=0 和 t=Tt=Tt=T 时的状态），但由于系统是时间演化的，通常采用初值表述。

  1. 内在结构与可积性
    这个方程组看似复杂，实际上具有丰富的数学结构，是完全可积系统（Liouville integrable system）的典型例子：

守恒量（首次积分）
将横向方程改写为全微分形式：

ddt[r2(ϕ˙+ω)]=0⟹r2(ϕ˙+ω)=常数=ωr02.\frac{d}{dt}\bigl[r^2(\dot{\phi} + \omega)\bigr] = 0 \quad\Longrightarrow\quad r^2(\dot{\phi} + \omega) = \text{常数} = \omega r_0^2. dtd​[r2(ϕ˙​+ω)]=0⟹r2(ϕ˙​+ω)=常数=ωr02​.

这给出一个守恒量（相当于惯性系中的角动量）。将其代入径向方程可得仅含 rrr 的二阶方程：

r¨=ω2r04r3=−ddr ⁣(ω2r042r2).\ddot{r} = \frac{\omega^2 r_0^4}{r^3} = -\frac{d}{dr}\!\left(\frac{\omega^2 r_0^4}{2r^2}\right). r¨=r3ω2r04​​=−drd​(2r2ω2r04​​).

进一步积分得到“能量守恒”：

12r˙2+ω2r042r2=ω2r022.\frac{1}{2}\dot{r}^2 + \frac{\omega^2 r_0^4}{2r^2} = \frac{\omega^2 r_0^2}{2}. 21​r˙2+2r2ω2r04​​=2ω2r02​​.

因此系统拥有两个对合的首次积分（角动量与能量），自由度数为2，属于刘维尔可积系统。解可以通过求积（quadrature）完全得到。
